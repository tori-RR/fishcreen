# fishcreen

一个轻量的 Windows 托盘程序，用于在双屏环境中自动降低副屏亮度并显示渐变黑幕：

- 主屏：LG TV（不做任何亮度操作）
- 目标副屏：Dell S2417DG，稳定设备 ID `DELA0E7`
- 日常模式：鼠标离开 Dell 1.5 秒后将亮度设为 `0`；全局鼠标静止 30 秒后开始黑幕，并在 30–35 秒间渐变至全黑
- 观影模式：鼠标离开 Dell 1.5 秒后立即覆盖纯黑遮罩，并将亮度设为 `0`
- 鼠标进入 Dell 后：移除遮罩，并恢复暗屏前的亮度
- 关闭自动暗屏、退出、休眠或注销时：尝试恢复亮度
- 托盘菜单可启用或关闭当前用户的开机自动启动

## 构建

在 PowerShell 中执行：

```powershell
.\build.ps1
```

生成文件：

```text
bin\fishcreen.exe
```

构建需要 Windows 自带的 .NET Framework 4.x 64 位 C# 编译器；程序当前编译为 x64。

## 使用

1. 双击运行 `fishcreen.exe`。
2. 程序首次启动默认使用日常模式，之后会记住上次选择。
3. 右键托盘图标可选择日常模式、观影模式或关闭自动暗屏。
4. 日常模式下，鼠标离开 Dell 约 1.5 秒只降低背光；鼠标静止 30 秒开始渐暗，到 35 秒完全变黑。
   - 30–35 秒渐变期间，任何鼠标移动都会立即取消黑幕。
   - 鼠标在 LG 上重新移动：黑幕消失，但 Dell 继续保持亮度 `0`。
   - 鼠标进入 Dell：黑幕消失，并恢复暗屏前的亮度。
5. 观影模式下，鼠标离开 Dell 约 1.5 秒就会降低背光并出现黑幕。
6. 将鼠标移动回 Dell，画面和原亮度会恢复。
7. 托盘菜单可开关“开机自动启动”、立即恢复副屏或退出。
8. 双击托盘图标可以在当前模式与关闭状态之间切换。

## 恢复保护

暗屏前的亮度会临时记录在：

```text
%LocalAppData%\fishcreen\brightness-recovery.txt
```

正常恢复后记录会被删除。如果程序异常退出，下次启动时会优先尝试恢复记录的亮度。

运行日志位于：

```text
%LocalAppData%\fishcreen\app.log
```

当前版本暂未包含设置窗口和全局快捷键；这些可以继续逐步增加。

“开机自动启动”默认关闭。启用后会写入当前用户的注册表启动项，不需要管理员权限：

```text
HKCU\Software\Microsoft\Windows\CurrentVersion\Run\fishcreen
```
